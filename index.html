VisuWeave - Dynamic Presentation System

Tech Stack: Python, Speech-to-Text (ASR), NLP, OpenCV, GANs, Reinforcement Learning

Built an **AI system** using NLP and Reinforcement Learning to **extract presentation concepts** in real-time (latency < 2s). It developed a multi-source image pipeline with **GAN layouts** at ~25 FPS, enabling live, canvas-based presentations where visuals animate with speech.

EcoVista - AI-Driven Waste Segregation

Tech Stack: Python, OpenCV, YOLOv8, IoT-enabled Hardware

Engineered an **energy-efficient hardware prototype** using **YOLOv8** that achieved **92% accuracy** in waste segregation across 4 categories. The system implemented CVAT-based litter detection and included an app-based reward system, which reduced improper disposal by **35%** during pilot tests.
